model_utils.generate_code_suggestion(input_text, suggestion_label, root)

Generiert Code-Vorschläge basierend auf dem Eingabetext.

Diese Funktion verwendet das geladene KI-Modell, um Code-Vorschläge basierend auf dem Eingabetext zu generieren. Sie aktualisiert die Benutzeroberfläche, um den generierten Code anzuzeigen.

Parameter:

input_text (str): Der Eingabetext, für den Code-Vorschläge generiert werden sollen. suggestion_label (tk.Label): Das Label, das den generierten Code anzeigt. root (tk.Tk): Das Hauptfenster der Anwendung.

Globale Variablen:

model_loaded (bool): Gibt an, ob das KI-Modell erfolgreich geladen wurde.

Ausnahmen:

Exception: Wird ausgelöst, wenn ein Fehler bei der Generierung des Codes auftritt.

model_utils.initialize_model(script_dir, local_model_path, suggestion_label, root)

Initialisiert das KI-Modell und den Tokenizer.

Diese Funktion lädt das KI-Modell und den Tokenizer von einem vorgegebenen Pfad und speichert sie lokal. Sie aktualisiert auch die Benutzeroberfläche, um den Ladezustand des Modells anzuzeigen.

Parameter:

script_dir (str): Der Verzeichnispfad des Skripts. local_model_path (str): Der Speicherpfad für das lokale Modell. suggestion_label (tk.Label): Das Label, das den Ladezustand des Modells anzeigt. root (tk.Tk): Das Hauptfenster der Anwendung.

Globale Variablen:

tokenizer: Der Tokenizer für das KI-Modell. model: Das geladene KI-Modell. model_loaded (bool): Gibt an, ob das KI-Modell erfolgreich geladen wurde.

Ausnahmen:

Exception: Wird ausgelöst, wenn ein Fehler beim Laden des Modells auftritt.
